# Halo Terminal

一款 Terminal 风格的 Halo 主题。

![screenshot](https://user-images.githubusercontent.com/27671436/203283319-32a7384f-7b46-4c9e-9ec7-4abb796fc7cf.png)

## 特性
- 支持明暗模式切换；
- 首页公告设置；
- 备案信息设置；
- 支持 Halo 内部的大部分页面，包括文章、页面、分类、标签、归档等。
- 支持 Halo 应用市场的部分内容管理插件，包括友情链接、瞬间。
